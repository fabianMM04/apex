# apex legends

## 1. Cloning the repository
```
git clone https://github.com/fabianMM04/apex.git
```
## 2. Install dependecies
```
cd apex
npm install
```
## 3. Run project
```
npm start 
```
## 4. Go to
```
http://localhost:4200/
```
